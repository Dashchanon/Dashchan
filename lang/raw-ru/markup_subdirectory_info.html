Шаблон используется для генерации пути по умолчанию при сохранении файлов через диалог конфигурации загрузки.

Специальный символ nullable может быть заключён в <>, что позволяет удалить эту часть из пути, если соответствующее значение отсутствует.

По умолчанию используется шаблон \c-<\b->\t.

На 2ch.hk/mobi/625219 шаблон будет заменён на dvach-mobi-625219.

На arhivach.org/117403 шаблон будет заменён на arhivach-117403: опциональная часть null- удалена, поскольку форум не имеет разделов.
